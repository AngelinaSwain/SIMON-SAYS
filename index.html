Simon Says Game

Press any key to start the game

Highest Score: 0

Rules

  • The game starts when the player presses any key on the keyboard.
  • At the beginning of each level, the game will flash a color. The sequence starts with a single color and increases by one additional color each level.
  • After the color is displayed, the player must reproduce the sequence by clicking on the colors in the correct order.
  • If the player correctly matches the sequence, they advance to the next level, else the game ends.
